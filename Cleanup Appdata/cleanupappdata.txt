Docker Not Running=Docker neběží
Because the docker service is not currently running, this module will prompt you to delete the appdata for any/all docker applications which are installed=Protože služba Docker momentálně neběží, tento modul ti nabídne možnost smazat složky appdata všech nainstalovaných Docker aplikací.

Deleted=Smazáno
The following appdata paths have been determined based upon your previously installed applications to be no longer in use Select which ones you would like to be deleted=Následující složky appdata byly na základě dříve nainstalovaných aplikací vyhodnoceny jako nepoužívané. Vyber, které chceš smazat.
Note that this module looks at all of your previously installed docker applications to determine which appdata folders are abandoned. If you have deleted the previously installed list (either through Community Applications or manually), then it will not display those result. IE: If there is not a record for the folder within the previously installed apps, then you will not be prompted to delete the folder.=Upozornění: Tento modul prochází všechny dříve nainstalované Docker aplikace a identifikuje opuštěné appdata složky. Pokud jsi seznam dříve nainstalovaných aplikací smazal (např. přes Community Applications nebo ručně), tyto složky se nezobrazí. Jinými slovy: pokud neexistuje záznam ve výpisu dříve nainstalovaných aplikací, složka nebude nabízena k odstranění.
In case of a severely misconfigured app previously installed, the entire contents of a share MAY be offered up to delete It is your responsibility to confirm all of the folders being offered up to delete If you inadvertently delete a share which you actually do want to keep due to your own mistakes in configuring an application, the deletion is not reversable and the author of the plugin will take no responsiblility=V případě vážně chybně nakonfigurované aplikace může být nabídnuto ke smazání celé sdílené úložiště. Je tvojí odpovědností zkontrolovat všechny nabízené složky. Pokud omylem smažeš sdílení, které sis ve skutečnosti chtěl ponechat, kvůli chybné konfiguraci aplikace, nelze to vrátit zpět a autor pluginu za to nenese žádnou odpovědnost.
